# CodeSwitch Project

## Overview
This project focuses on **handling code-switching in text**, where multiple languages appear in a single sentence.  
Using **Natural Language Processing (NLP) and Machine Learning techniques**, the project detects code-switched sentences, identifies the languages used, and analyzes them for further processing.  
The goal is to enable better understanding of multilingual text for applications like chatbots, social media analysis, and sentiment detection.

## Features
- Detects sentences that contain multiple languages
- Identifies the specific languages used in each sentence
- Preprocesses text using NLP techniques (tokenization, stopword removal)
- Machine learning model predicts language distribution for each sentence
- Provides easy-to-read outputs for testing new text samples

## Tech Stack
- **Python** – main programming language  
- **Pandas & NumPy** – data handling  
- **NLTK / SpaCy** – text preprocessing and tokenization  
- **Scikit-learn / ML models** – classification of code-switched text  
- **VS Code** – development environment  
- **Git & GitHub** – version control
